=== User Data Log ===

Author: Martin Nestorov
Author URI: https://github.com/mnestorov

== Description ==

User Data Log is a very simple WordPress plugin for log users access parameters like IP address, referrer, user agent, etc. to database.

== Install ==

Place the content of this repository into wp-content/plugins/userdatalog (you'll need to create this folder manually) and activate the plugin from your WordPress admin. (Plugins->Installed Plugins).

== Requirements ==

* PHP 5.4+
* Wordpress 5+
